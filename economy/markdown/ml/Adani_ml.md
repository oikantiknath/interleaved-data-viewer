ഇന്ത്യയിലെ ഗുജറാത്തിൽ നിന്നുള്ള ഒരു വ്യവസായിയാണ് ഗൗതം അദാനി ( ഇംഗ്ലീഷ് : Gautam Adani , ഗുജറാത്തി : ગૌતમ અદાણી ). അഹമ്മദാബാദ് ആസ്ഥാനമായുള്ള അദാനി ഗ്രൂപ്പ് എന്ന കമ്പനികളുടെ ഉടമസ്ഥനാണ് ഗൗതം അദാനി. [ 3 ]
